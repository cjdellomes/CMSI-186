CMSI 186
Collection of programs made for LMU CMSI 186 Computer Programming Lab
1. CountTheDays
Determines the number of days between two dates. Dates should be input as month, day year.
2. PiSolver
Calculates an estimated value for pi by simulating 1000 darts being thrown at a unit circle inside a unit square.
3. BirthdaySolver
Simulates a million trials to determine the probability that at least 2 people will share a birthday among an input number of people.
4. RandomIntegrator
Uses Monte Carlo integration to find the definite integral of input equations.
5. RiemannIntegrator
Uses Riemann Sum integration to find the definite integral of input equations.
6. MapSolver
Finds the appropriate color configuration for coloring a multiregional map using only 4 colors.
